## Apache Hive

`Apache Hive` 是基于Hadoop的一个数据仓库工具，可以将结构化的数据文件映射为一张数据库表。通过类SQL语句（`HiveQL`）快速实现简单的`MapReduce`统计，不必开发专门的`MapReduce`应用，十分适合数据仓库的统计分析。 

`Hive`适合用来对一段时间内的数据进行分析查询。Hive 不适合用来进行实时的查询，因为它需要很长时间才可以返回结果。
